# Sample SDTM-derived ctas input data

Generated from `pharmaversesdtm` domains (dm, lb, vs, rs_onco) using
[`Input_Labs()`](https://IMPALA-Consortium.github.io/ctasapp/reference/Input_Labs.md),
[`Input_VS()`](https://IMPALA-Consortium.github.io/ctasapp/reference/Input_VS.md),
[`Input_RS()`](https://IMPALA-Consortium.github.io/ctasapp/reference/Input_RS.md),
[`Input_BMI()`](https://IMPALA-Consortium.github.io/ctasapp/reference/Input_BMI.md),
and
[`combine_ctas_input()`](https://IMPALA-Consortium.github.io/ctasapp/reference/combine_ctas_input.md).
Contains range-normalized labs, ratio-missing labs, numeric vital signs,
categorical response data, and weight categories.

## Usage

``` r
sample_sdtm_data
```

## Format

A list with six elements:

- data:

  Data frame: subject_id, parameter_id, timepoint_1_name,
  timepoint_rank, result, timepoint_2_name, baseline

- subjects:

  Data frame: subject_id, site, country, region

- parameters:

  Data frame: parameter_id, parameter_name, parameter_category_1 through
  \_3, plus ctas config columns

- custom_timeseries:

  Empty data frame

- custom_reference_groups:

  Empty data frame

- queries:

  Data frame of simulated clinical queries (approx 15\\ of data points)
  generated by
  [`simulate_query_data()`](https://IMPALA-Consortium.github.io/ctasapp/reference/simulate_query_data.md)

## Source

Generated from `pharmaversesdtm` via `data-raw/sample_sdtm_data.R`
