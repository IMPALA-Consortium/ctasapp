# Get parameter type icon mapping from configuration

Returns a named list mapping parameter types to Font Awesome icon names.

## Usage

``` r
get_param_icons()
```

## Value

A named list (e.g. `list(range_normalized = "flask", ...)`), or `NULL`
if
[`apply_config()`](https://IMPALA-Consortium.github.io/ctasapp/reference/apply_config.md)
has not been called.
